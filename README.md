# Verbis - Gerçek Zamanlı Yapay Zeka Tabanlı Çeviri Uygulaması

Verbis, farklı dilleri konuşan insanların birbirleriyle iletişim kurabilmesini sağlayan yapay zeka destekli gerçek zamanlı bir çeviri uygulamasıdır.

## Özellikler

- Gerçek zamanlı konuşma çevirisi
- QR kod ile kolay oda paylaşımı
- Claude/OpenAI API entegrasyonu ile doğal çeviriler
- Birden fazla dil desteği
- Kullanıcı dostu arayüz

## Kurulum

### Gereksinimler
- Node.js 16+
- Docker ve Docker Compose (veritabanları için)

### Geliştirme

```bash
# Repoyu klonlayın
git clone https://github.com/cretique/verbis.git
cd verbis

# Veritabanlarını başlatın
docker-compose up -d

# Bağımlılıkları yükleyin
npm install

# Geliştirme modunda başlatın
npm run dev
```
